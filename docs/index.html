   debian-netsurf     

"NetSurf is a free, open source web browser. It is written in C and released under the GNU Public Licence version 2. NetSurf has its own layout and rendering engine entirely written from scratch. It is small and capable of handling many of the web standards in use today."    - NetSurf web page.

ewsdocker/debian-netsurf is a Docker container wrapper around the NetSurf browser.
